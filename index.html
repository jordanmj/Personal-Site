    Past Projects

    Crimeseekr

    Crimeseekr is a responsive, mobile-friendly web app designed to connect security professionals and civilians, improving collaboration and response times in critical situations. Users can report crimes, view incidents in their area, and help optimize security efforts for faster action and greater community safety.

    GitHub API

    This project is a RESTful API built with Flask that allows users to retrieve GitHub profile information and repository data. Users can access detailed profiles and a list of public repositories for any GitHub user. The API is designed to be lightweight and easy to integrate into applications that need to interact with GitHub data.

    VoIP Application

    This project is a Java-based VoIP application that enables real-time voice communication over the internet. It features a simple interface for connecting users and facilitates high-quality audio transmission, making it ideal for personal or small-scale professional use.

    Alan Compiler

    This project is a compiler for the Alan (Another Language) programming language, developed in C. It includes a scanner, parser, lexical analyzer, and code generator. The compiler translates Alan code into Java bytecode using Jasmin, allowing execution on the Java Virtual Machine (JVM). It handles syntax analysis, lexical processing, and efficient bytecode generation for executing Alan programs.

    Data Compressor

    This project is a Unix CLI tool that uses the Huffman algorithm for lossless file compression and decompression. Built in x86 Assembly, the data compressor leverages low-level programming techniques to achieve efficient file compression, providing a fast and compact solution for storing and transmitting data.

    Markdown Application

    This project is a full-stack web application built with React and GraphQL, allowing multiple users to collaborate on the same note in real-time. Users can write and edit notes in Markdown, with live output updates as changes are made. The app also enables users to save and share notes, making it a powerful tool for collaborative writing and note-taking.
